# Deep-Chrome-Dino
Simple implementation of a Deep Q Neural Network made in Pytorch that learns how to play the Google Chrome dinosaur running game.

For this project I took inspiration from [Python + PyTorch + Pygame Reinforcement Learning â€“ Train an AI to Play Snake](https://www.youtube.com/watch?v=L8ypSXwyBds)
